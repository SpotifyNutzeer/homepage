# Personal Homepage `paul.wtf` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build, containerize, and deploy a personal homepage at `paul.wtf` — German, Dark-only, in the linkhop liquid-glass design family.

**Architecture:** Astro 4+ static site, no UI framework islands. CSS tokens ported from linkhop. Multi-stage Docker build (node → nginx static). Plain Kubernetes manifests in `k8s/`, deployed via FluxCD (added in a separate follow-up).

**Tech Stack:** Astro, TypeScript, pnpm, Vanilla JS (mailto + clipboard), Sharp + Playwright (build-time linkhop screenshot), nginx, Docker, GitHub Actions, GHCR, Traefik, cert-manager, external-dns.

**Reference Spec:** `docs/superpowers/specs/2026-05-03-homepage-design.md` — read first if any task feels ambiguous.

**Test discipline:** Component/E2E tests are explicitly out of scope (see spec §Testing). Per-task verification = `pnpm astro check` + `pnpm build` + browser spot-check. Where relevant, a task includes a "Verify in browser" step with the exact thing to look at.

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: homepage
  namespace: homepage
spec:
  replicas: 1
  strategy:
    type: RollingUpdate

      maxSurge: 1

    matchLabels:
      app.kubernetes.io/name: homepage

    metadata:
      labels:
        app.kubernetes.io/name: homepage

      containers:
        - name: homepage
          image: ghcr.io/spotifynutzeer/homepage:v0.1.0
          imagePullPolicy: IfNotPresent
          ports:
            - name: http
              containerPort: 80
          resources:
            requests:
              cpu: 50m
              memory: 64Mi
            limits:
              cpu: 200m
              memory: 128Mi
          readinessProbe:
            httpGet:
              path: /
              port: http
            initialDelaySeconds: 2
            periodSeconds: 10
          livenessProbe:
            httpGet:
              path: /
              port: http
            initialDelaySeconds: 10
            periodSeconds: 30
          securityContext:
            allowPrivilegeEscalation: false
            readOnlyRootFilesystem: false
            runAsNonRoot: false
```

```yaml
    steps:
      - uses: actions/checkout@v4
```

```yaml
      - uses: docker/login-action@v3
        with:

          images: ghcr.io/spotifynutzeer/homepage

            type=ref,event=tag

        with:
          context: .
          platforms: linux/amd64
          push: true
          tags: ${{ steps.meta.outputs.tags }}
          labels: ${{ steps.meta.outputs.labels }}
          cache-from: type=gha
          cache-to: type=gha,mode=max
```
